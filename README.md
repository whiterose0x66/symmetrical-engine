# Sierpinski Triangle 
The Sierpinski Triangle is a famous self-similar fractal introduced by polish mathematician Wacław Sierpiński in 1915.

# There exist several process for constructing the Sierpinski triangle :
1) the basic construction :
   1) Start with an equilateral triangle.
   2)  Subdivide it into four smaller congruent equilateral triangles and remove the central triangle.
   3)   Repeat step 2 with each of the remaining smaller triangles infinitely.
